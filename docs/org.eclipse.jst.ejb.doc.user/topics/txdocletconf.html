Configuring XDoclet annotation support

To use XDoclet annotation support, or to create enterprise beans, XDoclet must be installed on your system and configured to work with the workbench.

More information on XDoclet can be found at http://xdoclet.sourceforge.net/xdoclet/index.html.

To configure XDoclet to work with the workbench, complete the following steps:

  1. Download and unzip xdoclet-bin-1.2.3.zip by navigating to http://xdoclet.sourceforge.net/xdoclet/install.html and following the download link.
  2. Click Window > Preferences .
  3. In the Preferences window, click XDoclet in the left pane.
  4. Check the Enable XDoclet Builder check box to turn on annotation-based artifact creation.
  5. Use the Browse button to locate the folder where you unzipped the XDoclet download (XDoclet Home).
  6. Select the Version of XDoclet that you have installed. In this case, select 1.2.3. If you downloaded an earlier version, XDoclet versions 1.2.1 and 1.2.2 are also supported.
  7. Click OK to save the preferences.
